Exercício Nº 1: Gestão das inscrições numa prova desportiva

Considera o ficheiro JSON com a lista dos atletas inscritos na ultra trail da Geira romana. Neste exercício, vais explorar a sua informação depois de o carregares num servidor Mongo.

Nesse sentido, tenta resolver as seguintes alíneas:

  1. Importa o ficheiro JSON no teu servidor Mongo: mongoimport --db geira --collection atletas --file inscritos.json;
  2. Lista todos os registos;
  3. Lista apenas o nome e a morada de cada atleta (projeção);
  4. Conta quantos atletas estão inscritos na "Corrida da Geira";
  5. Conta quantos atletas femininos estão inscritos na ultra;
  6. Lista o nome e a data de nascimento dos atletas da equipa "TURBULENTOS";

Exercício Nº 2: Base de dados de avaliações

Considera o ficheiro CSV com o resultado das avaliações de PRI2015. Neste exercício, vais transformá-lo, importá-lo para um servidor Mongo e explorá-lo realizando algumas queries.

Nesse sentido, tenta resolver as seguintes alíneas:

  1. Descarrega o ficheiro CSV e o conversor. Depois de analisares o conversor e perceberes o que faz, aplica-o de forma a obteres um ficheiro com uma lista de objetos JSON;
  2. Abre um terminal e coloca a correr uma sheel do MongoDB;
  3. Cria a base de dados "PRC2015": use <database>;
  4. Importa o ficheiro JSON obtido num servidor Mongo utilizando o comando mongoimport, para uma coleção a ser criada chamada "alunos";
  5. Lista as coleções da base de dados correntes: show collections;
  6. Lista os documentos na coleção alunos: db.alunos.find();
  7. Conta o número de registos na coleção alunos;
  8. Lista os registos na coleção alunos numa forma "mais bonita";
  9. Lista apenas o identificador e o nome de cada aluno (projeção);
  10. Lista os alunos com nota final superior a 14;
  11. Lista os alunos com nota final entre 10 e 14;

Exercício Nº 3: Base de dados de publicações

Considera o ficheiro CSV com as publicações do JCR. Neste exercício, vais importá-lo para um servidor Mongo e explorá-lo realizando algumas queries.

Nesse sentido, tenta resolver as seguintes alíneas:

  1. Importa o ficheiro JSON no teu servidor Mongo: mongoimport --db geira --collection atletas --file inscritos.json;
  2. Quantos registos tem esta nova coleção?
  3. Lista os registos (apenas id e título);
  4. Quantos artigos foram publicados em conferências? (inproceedings)
  5. Quantos artigos foram co-autorados com "Pedro Rangel Henriques"?
  6. Lista o título dos artigos co-autorados com "Ricardo Freitas" em 2013?
  7. Quantos artigos foram co-autorados com "Ricardo Freitas"?
