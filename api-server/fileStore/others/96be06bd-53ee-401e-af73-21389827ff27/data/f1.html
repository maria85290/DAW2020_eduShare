Exercício Nº 1: Soma de 2 inteiros

Pretende-se desenvolver uma micro-aplicação Web para somar 2 valores inteiros. Neste contexto, desenvolve as alíneas seguintes:

  1. Crie um formulário Web para a introdução de 2 valores inteiros;
  2. Crie uma pequena script em PHP que recebe os valores introduzidos e gera uma página Web com o resultado da soma daqueles valores;
  3. Crie 2 versões da script: uma usando o método POST e outra usando GET. Experimente as 2 e ganhe consciência para a diferença.

Exercício Nº 2: Soma de N inteiros

Pretende-se agora ampliar o âmbito da aplicação desenvolvida para somar até 5 valores inteiros. Neste contexto, desenvolve as alíneas seguintes:

  1. Crie um formulário Web para a introdução de 2 valores inteiros;
  2. Acrescenta-lhe dois botões, "+" e "-", para adicionar ou remover operandos;
  3. Cria uma script em JQUERY que altere o formulário de acordo com os cliques naqueles botões. O formulário nunca deverá ter mais de 5 operandos ou menos de 2;
  4. Crie uma pequena script em PHP que recebe os valores introduzidos e gera uma página Web com o resultado da soma daqueles valores;
  5. Crie 2 versões da script: uma usando o método POST e outra usando GET. Experimente as 2 e ganhe consciência para a diferença.

Exercício Nº 3: Registo de alunos

No contexto letivo de uma disciplina pretende-se desenvolver um pequeno aplicativo Web para registo dos alunos inscritos.

Para concretizar esse objetivo, desenvolva cada uma das seguintes alíneas:

  1. Crie um formulário Web para introdução dos dados do aluno, constituídos por nome, identificador (número atribuído pela instituição), email e, opcionalmente, o endereço de uma página Web; Use o método POST;
  2. Crie um novo formulário para o mesmo fim usando o método GET;
  3. Crie uma única script em PHP que recebe os dados de ambos os formulários e os grava num ficheiro XML (defina uma estrutura para o XML baseada numa lista de registos);
  4. Crie uma script PHP para listar o conteúdo do ficheiro XML numa página Web.

Exercício Nº 4: Registo de grupos alunos

No contexto letivo de uma disciplina pretende-se desenvolver um pequeno aplicativo Web para registo dos grupos de alunos que irão desenvolver projetos.

Para concretizar esse objetivo, desenvolva cada uma das seguintes alíneas:

  1. Crie um formulário Web para introdução dos dados dos vários elementos do grupo, constituídos por nome, identificador (número atribuído pela instituição), email e, opcionalmente, o endereço de uma página Web; Use o método POST;
  2. Crie um novo formulário para o mesmo fim usando o método GET;
  3. Crie uma única script em PHP que recebe os dados de ambos os formulários e os grava em dois ficheiros XML (defina uma estrutura para os documentos XML baseada numa lista de registos): um com a lista de grupos e outro com uma lista de alunos;
  4. Crie uma script PHP para listar o conteúdo dos 2 ficheiros XML numa página Web.

Exercício Nº 5: Registo de SIPs

Pretende-se desenvolver uma pequena aplicação Web para criação de SIPs.

Com esse objetivo, desenvolva as seguintes alíneas:

  1. Crie um formulário Web para receber os dados de um manifesto de acordo com o DTD desenvolvido nas aulas;
  2. Crie a respetiva script de tratamento da informação vai:
    • validar o conteúdo do manifesto;
    • compactar toda a informação relacionada num arquivo ZIP;
    • gerar uma página Web com um relatório descritivo deste processo.
