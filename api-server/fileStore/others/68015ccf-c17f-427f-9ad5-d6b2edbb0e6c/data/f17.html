Exercício Nº 1: Tabela Periódica dos Elementos

Considera a ontologia em formato RDF sobre a tabela periódica de elementos dosponível em cima e tenta responder às alíneas seguintes.

  1. Abre a ontologia no Protégé e realiza algumas queries exploratórias em SPARQL: que classes existem na ontologia, que propriedades aasociadas às classes principais;
  2. Especifica uma query SPARQL para responder à questão: Quantos elementos há na tabela periódica?
  3. Especifica uma query SPARQL para responder à questão: Qual o símbolo, o nome e o número atómico de cada elemento?
  4. Obtenha a última lista ordenada por número atómico.
  5. Especifica uma query SPARQL para responder à questão: Quais são os 3 elementos com maior número atómico?
  6. Cada elemento na tabela pertence a um grupo. Obtenha o número e o nome de cada grupo (há cerca de 20 grupos).
  7. Obtenha o nome, o símbolo e o número atómico dos elementos do grupo 18.
  8. Qual o número de elementos em cada grupo? (distribuição dos elementos por grupos)
  9. Quais os 5 grupos com mais elementos?
  10. Qual a média dos números atómicos dos elementos na tabela?
  11. Qual o maior número atómico na tabela?
  12. Qual o menor número atómico na tabela?
  13. Qual o nome do elemento com esse número atómico?
  14. Acrescenta à ontologia o conceito de molécula com as propriedades suficientes para adicionares algumas instâncias como: água (H2O), dióxido de carbono (CO2), amoníaco (NH3), metano (C2H6), glicose (C6H12O6), ácido súlfúrico (H2SO4). Foca-te na sua constituição e considera o seu nome, ponto de fusão e ponto de ebulição.

Exercício Nº 2: História de Portugal

Depois de descarregares a ontologia e a explorares um pouco no Protégé tenta responder com SPARQL às seguintes questões:

  1. Quantos triplos existem na Ontologia?
  2. Que classes estão definidas?
  3. Que propriedades tem a classe "Rei"?
  4. Quantos reis aparecem na ontologia?
  5. Calcula uma tabela com o seu nome, data de nascimento e cognome.
  6. Acrescenta à tabela anterior a dinastia em que cada rei reinou.
  7. Qual a distribuição de reis pelas 4 dinastias?
  8. Lista os decobrimentos (sua descrição) por ordem cronológica.
  9. Lista as várias conquistas, nome e data, juntamento com o nome que reinava no momento.
  10. Calcula uma tabela com o nome, data de nascimento e número de mandatos de todos os presidentes portugueses.
  11. Quantos mandatos teve o presidente Sidónio Pais? Em que datas iniciaram e terminaram esses mandatos?
  12. Quais os nomes dos partidos políticos presentes na ontologia?
  13. Qual a distribuição dos militantes por cada partido político?
  14. Qual o partido com maior número de presidentes militantes?
