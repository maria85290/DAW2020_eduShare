WebServices com SOAP

Processamento e Representação de Conhecimento

O objectivo principal desta ficha é exercitar o aluno na criação de clientes SOAP para serviços disponíveis na Internet.

Especificamente, nesta ficha, o foco será dado à criação de clientes que executam na linha de comando e à independência da linguagem, será pedido ao aluno que crie os clientes em duas linguagens: Perl e PHP.

Exercícios:

  1. Números Aleatórios
  2. O Dia do teu Nascimento
  3. Temperaturas

Exercício Nº 1: Números Aleatórios

Foi criado um serviço que gera um número aleatório entre um limite mínimo e um limite máximo que devem ser passados como parâmetros.

Os dados da API do serviço são os seguintes:

  Ponto de acesso: http://epl.di.uminho.pt/jcrbin/WebServices/nusoap-0.9.5/Aleatorio/serv-aleatorio.php
  API: geraNumAleatorio( int min, int max )

Desenvolva as seguintes alíneas:

  1. crie um cliente em PHP e teste-o com alguns valores;
  2. altere o cliente desenvolvido anteriormente para gerar uma lista de 100 valores aleatórios entre 1 e 1000;
  3. crie um cliente Perl para o mesmo serviço que funcione na linha de comando.

Exercício Nº 2: O Dia do teu Nascimento

Foi criado um serviço que dada uma data calcula o respetivo dia da semana.

Os dados da API do serviço são os seguintes:

  Ponto de acesso: http://epl.di.uminho.pt/jcrbin/WebServices/nusoap-0.9.5/DayOfBirth/serv-dayofbirth.php
  API: DayOfBirth( int year, int month, int day )

Desenvolva as seguintes alíneas:

  1. crie um cliente em PHP e teste-o com alguns valores, por exemplo, a data do seu nascimento;
  2. altere o seu cliente para carregar um ficheiro XML constituído por pares (nome, data de nascimento) e que gera uma lista de pares (nome, data de nascimento, dia da semana);
  3. crie um cliente Perl para o mesmo serviço que funcione na linha de comando.

Exercício Nº 3: Temperaturas

Neste exercício vais criar um serviço de conversão de temperaturas.

Desenvolve as seguintes alíneas:

  1. cria um servidor em PHP que relize as conversões de temperaturas entre graus Celsius, Fahreneit e Kelvin;
  2. cria 2 clientes, um em Perl e outro em PHP para o serviço;
  3. testa tudo com vários exemplos;
  4. cria um novo cliente em linguagem à tua escolha que produz uma tabela de temperaturas em que o domínio em Celsius vai de -300º a +300º.
